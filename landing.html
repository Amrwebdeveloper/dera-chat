واجهة شات بوت عربية بتصميم Glassmorphism، بـ٦ قوالب ردود (نص، صورة، خريطة، كاروسيل، سلايدر، ألبوم)، Lightbox، وتحكّم كامل في الألوان والهوية والمكان — كل ده محلي وبدون أي مكتبات خارجية.

كيفية الاستخدام

الويدجت شغّال فعلاً في ركن الصفحة — اضغط الزر الدائري ↙

كل اللي تحتاجه — جاهز

قوالب جاهزة، تحكّم كامل، وأداء محلي بالكامل.

💬

٦ قوالب ردود

نص (تأثير كتابة)، صورة + تعليق، خريطة (embed)، كاروسيل، سلايدر تلقائي، وألبوم شبكة بـ«+N».

🔍

Lightbox

تكبير بالنقر/العجلة/الأزرار + تحريك بالسحب، تنقّل بالكيبورد، وحبس تركيز — يدعم الاتجاهين.

🎨

ثيم كامل

لون، حجم، شكل (radius)، خط، ومكان — كله من مكان واحد عبر متغيّرات CSS تتحدّث فورًا.

🪪

هوية البوت

اسم، حالة اتصال، وصورة (أو أيقونة) — تتحكّم فيها لحظيًا.

🎛️

مفاتيح تشغيل

أظهر/أخفِ أي عنصر (الإرفاق، ملء الشاشة، الشرائح، السجل…) بـtrue/false.

↔️

RTL و LTR

السحب، الأسهم، والمواضع تنعكس منطقيًا — يعمل بكفاءة في الاتجاهين.

🚀

زر عائم + إرسال

تحكّم في شكل وأيقونة ومكان الزر العائم وزر الإرسال.

🔒

آمن محليًا

كل النصوص عبر textContent، iframes موثوقة فقط، وبدون أي CDN.

كيفية الاستخدام — ٣ خطوات

انسخ المجلد، أضف الأكواد، وخصّص. خلاص.

  1. 1

    انسخ مجلد assets/ إلى مشروعك

    يحتوي على الخطوط، الـCSS، والـJS — كله محلي بدون أي اعتماد خارجي.

  2. 2

    أضف الـCSS و الـJS في صفحتك

    الترتيب مهم: الإعدادات والوحدات قبل التطبيق. الويدجت يظهر تلقائيًا.

    <!-- في <head> -->
<link rel="stylesheet" href="assets/css/styles.css">
<link rel="stylesheet" href="assets/css/components.css">

<!-- قبل نهاية <body> -->
<script src="assets/js/config.js"></script>
<script src="assets/js/lightbox.js"></script>
<script src="assets/js/carousel.js"></script>
<script src="assets/js/slider.js"></script>
<script src="assets/js/messages.js"></script>
<script src="assets/js/bot.js"></script>
<script src="assets/js/app.js"></script>

    لاحظ: انسخ أيضًا بنية HTML الخاصة بالويدجت من index.html (تُحدَّث لاحقًا لتصبح new ChatWidget() بسطر واحد).

  3. 3

    خصّصه على ذوقك

    كل التخصيص من config.js أو لحظيًا عبر دوال MedChat:

    // اللون، الحجم، الشكل، المكان
MedChat.applyTheme({
  theme:  { primary: '#e11d48', accent: '#f59e0b', radius: '18px' },
  layout: { side: 'right', fabBottom: '2rem' },
  fab:    { icon: 'headset', shape: 'rounded' },
  send:   { icon: 'arrow', rotate: false },
});

// هوية البوت
MedChat.applyIdentity({ name: 'عيادتي', status: 'يرد خلال دقائق', avatar: 'logo.png' });

// مفاتيح تشغيل (true/false)
MedChat.applyFeatures({ history: false, attachButton: true });

قوالب الردود (للربط بالباك-إند)

البوت يختار القالب حسب نوع الإجابة ويُرجع كائن { type, ... }. للربط بخادم حقيقي، اجعل getResponse في bot.js تنادي الـAPI وتُعيد نفس الشكل. 

// نص
{ type: 'text', text: 'مرحباً 👋' }

// صورة + تعليق (Lightbox عند النقر)
{ type: 'image', text: 'نصيحة اليوم', image: { src: 'tip.png', alt: 'نصيحة', caption: 'اشرب ماءً كافياً' } }

// خريطة / تضمين (iframe موثوق)
{ type: 'embed', text: 'موقعنا', embed: { src: 'https://www.openstreetmap.org/export/embed.html?...', title: 'الموقع', caption: 'العيادة', openUrl: 'https://...' } }

// كاروسيل (بطاقات بجوار بعض)
{ type: 'carousel', text: 'معرض', images: [ { src, alt, caption }, /* ... */ ] }

// سلايدر (واحدة تلو الأخرى + تشغيل تلقائي)
{ type: 'slider', text: 'عرض', images: [ /* ... */ ] }

// ألبوم شبكة (بحد أقصى ثم +N)
{ type: 'gallery', text: 'الألبوم', images: [ /* ... كثير ... */ ] }
