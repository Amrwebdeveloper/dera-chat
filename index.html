Get started — two lines

Include the bundle, then create an instance with your options.

<script src="dera-chat.min.js"></script>
<script>
  const chat = new DeraChat({
    identity: { name: 'Acme Support', avatar: 'logo.png' },
    theme:    { primary: '#3b82f6', accent: '#2dd4bf', radius: '20px' },
    layout:   { side: 'right' },
    fab:      { icon: 'headset', shape: 'rounded' },

    // connect your backend (async), or use a `responses` map
    onMessage: async (text) => {
      const r = await fetch('/api/chat', { method:'POST', body: JSON.stringify({ text }) });
      return await r.json();   // { type:'text'|'image'|'carousel'|... }
    },
  });

  chat.on('message', (e) => console.log('user said:', e.text));
  chat.on('reply',   (e) => console.log('bot replied:', e.response));
</script>

Reply templates

Your onMessage returns one of these shapes; DeraChat renders it.

{ type: 'text', text: 'Hello 👋' }

{ type: 'image', text: 'Tip', image: { src: 'tip.png', alt: 'Tip', caption: 'Stay hydrated' } }

{ type: 'embed', text: 'Find us', embed: { src: 'https://…/embed', caption: 'HQ', openUrl: 'https://…' } }

{ type: 'carousel', text: 'Gallery', images: [ { src, alt, caption }, … ] }

{ type: 'slider',   text: 'Slideshow', images: [ … ] }

{ type: 'gallery',  text: 'Album', images: [ …many… ] }   // shows N tiles + “+N”
